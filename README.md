1. Project: Automation Testing Framework for bstackdemo.com using Selenium, Java & TestNG

2. Objective: The objective of this project is to automate the functional testing of the bstackdemo.com e-commerce website using Selenium WebDriver and TestNG. The framework aims to validate key user workflows such as login, product selection, cart management, and checkout. It ensures application reliability by executing positive and negative test scenarios. The project also focuses on building a scalable and maintainable automation framework using the Page Object Model design pattern.

3. Technology used: 
• Programming Language: Java 
• Web Automation: Selenium WebDriver
• Test Framework & Assertions: TestNG 
• Build & Dependency Management: Maven 
• Design Pattern: Page Object Model (POM)
• Logging: Log4j2

4. Reporting:-
• TestNG default reports 
• Extent Reports
